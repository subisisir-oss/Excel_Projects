Bank Loan Performance Analytics

Project: Bank Loan Performance Dashboard — Excel
Author: Subhisha C

Overview

This Excel dashboard analyzes bank loan data to uncover performance insights across branches, loan products, repayment behavior, and client activity. The dashboard provides interactive visuals and metrics to help stakeholders evaluate portfolio health, track customer retention, and monitor loan servicing efficiency.

Key Features

Yearly client retention analysis

Delinquency vs default loan rate comparison

Product-wise profitability and loan distribution

Branch-level on-time repayment behavior

Trend of new client acquisition by year

KPI summary for client counts, recovery rate, and interest income

Interactive slicers for year, branch, and account filtering

Clean and professional financial reporting layout
